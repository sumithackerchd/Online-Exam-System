# 📝 Online Exam System

A complete Online Exam System built using Flask, SQLite and Bootstrap.

## 🚀 Features

- 🔐 Admin Login
- ➕ Add Questions
- 📋 View Questions
- 🗑 Delete Questions
- 📝 Start Exam
- ⏱ 60 Second Countdown Timer
- ✅ MCQ Based Exam
- 📊 Automatic Score Calculation
- 🎉 Result Page
- 💾 SQLite Database
- 🎨 Responsive Bootstrap UI

## 🛠 Technology Used

- Python
- Flask
- SQLite
- Bootstrap 5
- HTML5
- CSS3

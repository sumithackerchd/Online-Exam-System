# 📝 Online Exam System

A complete Online Exam System built using Python Flask, SQLite and Bootstrap.

## 🚀 Features

* 🔐 Admin Login System
* ➕ Add Questions
* 📋 View Questions
* 🗑 Delete Questions
* 📝 Start Exam
* ✅ MCQ Based Exam
* ⏱ 60 Second Countdown Timer
* 📊 Automatic Score Calculation
* 🎉 Result Page
* 💾 SQLite Database
* 🎨 Bootstrap Responsive UI

## 🛠 Technology Used

* Python
* Flask
* SQLite
* Bootstrap 5
* HTML5
* CSS3
